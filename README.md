# MrSobolle Theme

Object-oriented WordPress portfolio theme for [mrsobolle.com](https://mrsobolle.com).

This theme is built as a practical demonstration of modern WordPress theme architecture: clean PHP classes, separated theme services, structured templates, responsive frontend assets, and a foundation for SEO-oriented development.

## Purpose

The project is intended to serve both as the active theme for mrsobolle.com and as a public portfolio example for WordPress/PHP development.

## Current Features

- Object-oriented theme structure.
- Custom autoloader.
- Theme setup service.
- Asset registration service.
- Basic frontend templates.
- Reusable template parts.
- Structured data provider classes.
- Theme screenshot and `theme.json`.

## Tech Stack

- WordPress
- PHP
- HTML
- CSS
- JavaScript
- jQuery-ready frontend approach

## Status

Early development. The architecture and public pages are being built step by step.
